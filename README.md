# NightShift: Underground

Arcade street racing game set in a neon-lit underground city.
Inspired by classic NFS-style gameplay: illegal races, police chases,
reputation system and car customization.

## Core Features
- Arcade driving physics
- Drift system
- Rival AI
- Police pursuit & heat system
- Reputation & progression
- Illegal events & night races

## Tech Stack
- Unity (C#)
- Git + GitHub
- VS Code
